TypeScript nima?

TypeScript - bu JavaScript tilining ustiga qurilgan superset bo'lib, kuchli tiplash imkoniyatlarini taqdim etadi. U Microsoft tomonidan ishlab chiqilgan.

Afzalliklari

  • Statik tiplash: Kodingizda xatolarni erta aniqlash imkonini beradi.
  • OOP qo'llab-quvvatlashi: Class lar, interfeyslar va meros olish kabi imkoniyatlarni taqdim etadi.
  • Katta loyihalar uchun qulaylik: Kodni yaxshiroq tashkil qilish va o'qiluvchanlikni oshiradi.
  • JavaScript bilan to'liq moslik: TypeScript kodini JavaScript-ga kompilyatsiya qilish mumkin.

Misol

Quyida TypeScript kodining oddiy namunasi keltirilgan:

function salomBer(ism: string): string {
    return `Salom, ${ism}!`;
}
console.log(salomBer("Doston"));

Xulosa

TypeScript dasturchilarga kod yozishni yanada samarali va ishonchli qilishga yordam beradi. Bu tilni o'rganish JavaScript asosida katta loyihalar bilan ishlashni osonlashtiradi.
